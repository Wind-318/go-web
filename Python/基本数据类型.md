下面介绍 Python 中的 List（列表）、tuple（元组）、dictionary（字典）、set（集合）类型
- ## List 与 Tuple
```python
# List 是一种可以动态增长的数组，可以随时添加删除其中的元素

# 创建一个拥有两个元素的 list
list1 = [1,1]
# 将 list1 赋值一份，赋值给 list2
list2 = list1.copy()
print(list2)
# 对 list1 追加元素
list1.append("asd")
print(list1)
# 统计 list1 中 “1” 的数量
print(list1.count(1))
# 搜索 0 到 3 的索引区间，返回 “1” 这个元素的下标
print(list1.index(1, 0, 3))
# 在索引 2 处插入 “qwe” 元素
list1.insert(2, "qwe")
# 按下标索引，删除 1 索引处的元素
list1.pop(1)
# 删除最后一个元素
list1.pop()
# 按元素寻找，删除这个元素
list1.remove("asd")
# 对 list1 进行翻转
list1.reverse()
print(list1)
# 清空 list1
list1.clear()
print(list1)
# 对 list2 排序(list 中的元素本身要能比较)
list2.sort()



# tuple 与 List 相似，不过元素不能修改，所以没有 append 和 pop 等方法
# 如果 tuple 中包含 list，当 list 中的元素变化时，tuple 仍然指向此 list，所以 tuple 没有改变，改变的是 list 指向的元素

# 示例：
tuple = (1,2, [3,4])
tuple[2][0] = 5
tuple[2][1] = 6
print(tuple)
```

- ## Dictionary
```python
# Python 中的 dict 在某些语言中叫 map，使用键值对存储(key-value)
map = {"name":"龙套乙"}
# 复制一个 map
temp = map.copy()
# 返回 key 对应的值，若没有则返回默认设置的值
map.get("asd",13)
# 查看 map 的所有键（key）
print(map.keys())
# 删除 key
map.pop("name")
# 插入 key，若已经有则返回对应值，若没有则返回设置的默认值
print(map.setdefault("age", 18))
# 查找 map 中的所有值（value），并返回
print(map.values())
# 清空
map.clear()
```

- ## Set
```Python
# set 存储 key，而且不能重复
# 添加一个元素
set.add(value)

# 显示一个集合和其它集合的不同，返回这个新的集合
set.difference(self, other)

# 从这个集合中移除其它集合中的 key
set.difference_update(self, *other)

# 从集合中移除一个元素
set.remove(value)
```
