# 网络层
- ## IP 协议
  - IP 是分配给每个用户设备的标签，IP 地址保证了每个设备的唯一性，常用 IP 地址分为 **IPv4** 和 **IPv6** 两种。![](IPv4数据报.jpg)
  - **版本**： IPv4 或 IPv6
  - **首部长度**：4 位
  - **数据报长度**：首部长度加数据长度
  - **寿命（TTL）**：用于确保数据报不会永远在网络中循环。寿命为 0 时丢弃数据报
  - **上层协议**：指明数据应交给哪一个协议处理
- ## 网际控制报文协议 ICPM
  - 可以用来确认数据是否成功传输和丢失的原因
  - ICPM 基于 IP 协议工作，但属于网络层协议
  - ICPM 协议类型分为两种
    - 查询报文
    - 差错报文
  - ### Ping
    - 作用：用于测试两台主机间连通性
    - 过程：主机向另一主机发送请求，目标主机回复应答，主机得到数据往返时间等信息，显示出来
