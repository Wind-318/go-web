## 二分查找
二分查找是一种对**有序序列**进行查找的算法。
- C++模板
```C++
// 待查找：vector<int> arr
// 目标值：target
int left = 0;
int right = arr.size();
while (left + 1 < right) {
    mid = left + (right - left) / 2;
    if (arr[mid] == target) {
        // 找到了
    } else if (arr[mid] < target) {
        // 中间值小于目标值，将查找区间变为右半部分
        left = mid;
    } else {
        // 中间值大于目标值，将查找区间变为左半部分
        right = mid;
    }
}
```
